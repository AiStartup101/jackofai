# JackOfAI Static HTML Website

This is a standalone HTML version of the JackOfAI website, converted from the React application.

## Files

- `index.html` - Complete standalone HTML file with embedded CSS and JavaScript
- `logo.jpeg` - Company logo image

## Usage

Simply open `index.html` in any web browser. No build process or server required!

The website includes:
- All content from the original React landing page
- Responsive design for mobile and desktop
- Interactive features:
  - Mobile menu toggle
  - Newsletter signup form
  - Email copy functionality
  - Founder chat modal
  - Smooth scrolling navigation

## Features

- **Self-contained**: All CSS and JavaScript is embedded in the HTML file
- **No dependencies**: Works without any build tools or frameworks
- **Responsive**: Mobile-friendly design
- **Modern styling**: Glass morphism effects and gradient backgrounds
- **Interactive**: All interactive features work with vanilla JavaScript

## Notes

- The logo image (`logo.jpeg`) must be in the same directory as `index.html`
- External links (LinkedIn, Instagram, etc.) point to the actual social media profiles
- Email links use `mailto:` protocol
- WhatsApp links open in WhatsApp app/web
