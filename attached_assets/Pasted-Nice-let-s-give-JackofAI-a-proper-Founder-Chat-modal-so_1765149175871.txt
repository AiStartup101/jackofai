1️⃣ Add modal state to HomePage

In app/page.tsx, inside the HomePage component (right after export default function HomePage() {), add:

const [isFounderChatOpen, setIsFounderChatOpen] = React.useState(false);


⸻

2️⃣ Create a “Chat with Founder” trigger button

Wherever “Chat with the Founder” should appear (for example in the Contact section or Investors section), instead of using SecondaryButton with an href, add this button:

<button
  onClick={() => setIsFounderChatOpen(true)}
  className="inline-flex items-center justify-center rounded-full border border-white/20 px-5 py-2.5 text-sm font-semibold text-slate-100 hover:bg-white/5 transition-colors"
>
  Chat with the Founder
</button>

You can keep other WhatsApp / email buttons as links — this one is just to open the modal.
